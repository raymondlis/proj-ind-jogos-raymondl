# Projeto Intro Unity - Survival Game

**Nome:** Raymond Lisbona

**Nome do Jogo:** MetalPipe

## Descrição do Jogo
Este é um jogo de sobrevivência onde o objetivo é durar o máximo de tempo possível. O jogador deve desviar dos inimigos que se movem de forma constante e rebatem pelas paredes.

### Controles
- **Teclado:** Movimentação via setas ou teclas WASD.
- **Gamepad:** Suporte total para controles (Analógico esquerdo).

### Mecânicas Principais
- **Sistema de Vidas:** Comece com 3 vidas. Colidir com inimigos retira uma vida. O jogo acaba quando as vidas chegam a zero.
- **Interface (UI):** HUD completo com cronômetro de tempo sobrevivido e contador de vidas.
- **Menu e Transições:** Menu principal, reinicialização rápida e retorno ao menu após o fim de jogo.
- **Audio:** Música de fundo em loop para maior imersão.
